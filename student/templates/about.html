About MarkX

MarkX is a web application made in Django (a web framework of Python) for managing the academic results of the students.

It can perform CRUD (Create, Read, Update and Delete) operations on the database. It uses SQLite as the backend RDBMS.

It features an easy to use admin interface to create and edit entries in the database.
